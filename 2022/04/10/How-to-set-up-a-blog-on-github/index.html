step5:
Install Hexo;
You can use ‘npm’ command after installing Node.js; Next:

Install hexo dependencies:

step6: Creat a new folder to put your blog files and use following command to initialize your project;

step7:
Configure SSH key:

Enter the file in which to save the key (/home/kali/.ssh/id_rsa.pub).
Copy the file content to paste into SSH and GPG keys on your account personal setting center.

step8:
Configure account and password.

step9:
Upload your blog to github.

Upload to github.
